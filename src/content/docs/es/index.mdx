El Proyecto App Fair es una organización benéfica sin fines de lucro dedicada a la creación de aplicaciones de software móvil para el bien público. El proyecto fue fundado en 2022 como un nexo mundial para que se construyan, distribuyan y mantengan aplicaciones gratuitas.

Las aplicaciones de App Fair no cuestan nada y no contienen tecnología publicitaria incorporada ni tecnologías de seguimiento no consensuadas. No intentan monetizar la atención del usuario de ninguna manera. Se crean a partir de software gratuito y de código abierto y se financian únicamente con donaciones y subvenciones.

Nuestro primer proyecto es lanzar el mercado de aplicaciones App Fair para iPhone y iPad en la Unión Europea en 2026 para distribuir estas aplicaciones lideradas por la comunidad. La aplicación del mercado App Fair se utilizará para descubrir, revisar, instalar y actualizar las aplicaciones de nuestro catálogo.
